# Dropbox Data Engineering Interview Gym — v5

This version makes all SQL interview questions runnable, including DBX-QSQL quick-fire questions that were previously rubric-only. Every SQL problem now has setup SQL, table schemas, sample data, expected output, and automatic result comparison.

Deploy by copying these files to the GitHub Pages repo root:

- index.html
- style.css
- app.js
- problems.js
- standalone.html
- README_DEPLOY.md

Then commit and push.
